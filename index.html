OSRM Distance

Set of scripts to calculate distance tables between sets of points in csv format relying on computation made by an existing local OSRM installation.

Demo website makes use of JS Pivot Table for the presentation, and Csv2Json for the transformation between output csv and a json format for the pivot.

In the repository there are a QUICKSTART doc discussing the creation of a local OSRM instance and a README with notes about the script.
